## Описание проекта:
Интерактивная страница с возможностью авторизации и регистрации пользователей, редактирования профиля, добавления и удаления фотографии, проставки лайков.

## Технологии, используемые в проекте
* HTML
* CSS (Flex, Grid, BEM)
* JavaScript
* React
* Node.js
* Express
* Mongoose
